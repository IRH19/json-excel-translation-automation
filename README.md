This project automates the localization workflow for web applications.

## How it works:
1. **JSON to Excel:** Converts English JSON keys/values into a clean Excel sheet for translators.
2. **Excel to JSON:** Takes the translated Excel file and reconstructs the JSON structure in the target language (Bahasa Melayu).

## Tech Stack:
- Python (Pandas/Openpyxl)
- PowerShell (for batch processing)
